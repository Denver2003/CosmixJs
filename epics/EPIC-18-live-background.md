# EPIC-18 Live Background

## Goal
Добавить живой космический фон за стаканом: статичное изображение + слой звёздных частиц.

## Scope
- Фоновый ассет 1024×2048 (чёрный placeholder)
- Отрисовка фона за стаканом, привязка к высоте стакана
- Звёздные частицы (лёгкая симуляция)

## Out of scope
- Художественные финальные ассеты
- Сложные шейдерные эффекты

## Done when
- Фон отображается за стаканом и масштабируется по высоте
- Звёзды движутся/мерцают без заметной нагрузки
