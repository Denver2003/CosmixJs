# EPIC-19 Level UI Overhaul

## Goal
Перевести визуальный стакан и HUD на новый арт-слой: PNG-стакан поверх сцены + обновленный шрифт и UI, постепенно заменяя текущие элементы по одному.

## Scope
- Базовый ассет стакана `glass_frame.png` (512x768) в `assets/levelUI/`.
- Удаление текущей отрисовки стакана кодом (только визуально), физика без изменений.
- Поэтапная замена UI-элементов на canvas-отрисовку и новый шрифт.

## Out of scope
- Полный редизайн всех экранов и шопа.
- Сложные VFX и шейдеры.

## Done when
- Стакан отображается через PNG-оверлей, старый визуальный рендер отключен.
- HUD использует новый шрифт и обновленные элементы (очки, монеты, прогресс уровня).
- Обновления выполняются поэтапно, по одному элементу за шаг.
