# EPIC-10 App Shell / Windows

## Goal
Базовая структура приложения: Home / Game / Shop / Settings / Pause / GameOver / Leaderboards.

## Scope
- Router / Screen Manager
- Overlay система (Pause, GameOver, dialogs)
- UI layout + safe area
- Back/ESC навигация

## Out of scope
- Покупки, реклама, облако

## Done when
- Все окна есть и стабильно переключаются
- Игра не ломается при уходе/возврате
