# EPIC-14.2 Cloud Saves

## Goal
Синхронизация прогресса между устройствами.

## Scope
- coins
- upgrades
- consumables inventory
- record (all-time best)
- (опционально) user settings audio
- Сохранение: game over, покупка апгрейда, покупка монет, смена настроек, периодически
- Fallback localStorage

## Out of scope
- Реклама/покупки/лидерборды

## Done when
- Прогресс синхронизируется
- Есть fallback на localStorage
