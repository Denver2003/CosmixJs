# EPIC-20 Canvas UI Overhaul

## Goal
Перенести весь UI (экраны, меню, оверлеи) на отрисовку в канвасе, без DOM-слоев.

## Scope
- HUD, меню паузы, Game Over, Home/Shop/Settings/Leaderboards — все в canvas.
- Общий роутинг экранов остается, но визуал отрисовывается в canvas.
- Единый шрифт/стили для UI (RussoOne).

## Out of scope
- Полная переработка логики игры или шопа.
- Подключение новых ассетов сверх текущих.

## Done when
- UI экранов/оверлеев рисуется в канвасе.
- DOM-слои используются только как контейнеры, без визуального UI.
